# ControllerUsingDCMotor

< 개발환경 >
- Window
- Arduino

< 동작 시나리오 >
- 거리센서를 통해 사용자가 특정 거리안에 들어오면 거중기를 들어올린다.
- 물체가 특정 거리만끔 올라오면 3초간 정지한다.
- 3초 이후 물체는 바닥까지 이동한다.
- 거중기는 다시 사용자가 특정 거리안에 올때까지 대기한다.
